Здравствуйте! Это мой проект 'Телефонная книга', выполненный на основе библиотек openpyxl и pandas. 
Хранение данных, экспорт и импорт осуществляется с помощью таблиц exсel.
При добавлении пользователем новой записи через консоль формируется ID по номеру строки.
На данном этапе не релизовано добавление ID при импорте файла пользователем, ячейка ID остаётся пустая.
